# 刷题记忆工具

一个本地刷题工具，用来把课程题库整理成可练习、可预览、可导出缓存的网页题库。

当前支持：

- `.docx` / `.doc` / `.pdf` 导入
- 单选题、多选题、判断题、填空题、简答题
- 顺序练习 / 乱序练习
- 错题本
- 全部预览
- 章节 / 知识点 / 原题号 / 文件名定位
- 本地缓存、缓存导入和导出
- 单片机固定模拟考试
- 自定义模拟组卷

## 启动方式

推荐用本地服务启动：

```powershell
python server.py
```

然后访问：

```text
http://127.0.0.1:5175
```

`.doc` 和 `.pdf` 都需要通过本地服务导入。直接打开 `index.html` 时，浏览器只能较可靠地读取 `.docx`。

## PDF 支持

PDF 导入依赖 PyMuPDF：

```powershell
python -m pip install -r requirements.txt
```

如果没有安装 PyMuPDF，导入 PDF 时会显示安装提示，不会静默生成乱码题库。文本型 PDF 会优先抽取文字；如果页面没有可抽取文字但包含图片，会尝试把页面作为图片保留到题库内容中。

## 题库格式

支持显式答案：

```text
1. 马克思主义有（ ）个基本组成部分。
A. 一
B. 二
C. 三
D. 四
答案：C
```

也支持答案嵌在题干括号里的客观题：

```text
1. 马克思主义有（ C ）个基本组成部分。
A. 一
B. 二
C. 三
D. 四
```

多选题会保存多个答案，练习和考试中需要全选正确才算对：

```text
1. 马克思主义理论体系的三个主要组成部分是（BCD）。
A. 马克思主义政治学
B. 马克思主义政治经济学
C. 科学社会主义
D. 马克思主义哲学
```

判断题支持：

```text
1. 马克思主义政治经济学是对英国古典政治经济学的继承和发展。（√）
```

填空题和简答题支持 `答案：` / `参考答案：` / `正确答案：`，练习时显示参考答案，由用户自己标记答对或答错；模拟考试中暂不自动计分。

## 模拟组卷

单片机模拟考试保留固定策略入口。

自定义模拟组卷支持分别设置单选、多选、判断、填空、简答数量；数量留空表示该题型全选。抽题时优先按章节均分，没有章节时按知识点均分，都没有时从题型题库中随机抽取。当前只计算单选、多选和判断分数，填空和简答只展示参考答案。

## 本地验收建议

更新后建议按下面顺序快速验收：

1. 用本地服务打开页面，导入 `马原21套题有答案版(1) (1)(4).pdf`，确认没有乱码，单选和多选能进入题库。
2. 导入 `2026春期末客观题题库《马克思主义基本原理》.docx`，确认单选、多选、判断均能识别。
3. 导入 `单片机填空题.docx`，确认填空题的代码换行保留，`stdio.h`、`FA` 这类答案不会被误判为判断题。
4. 在练习、全部预览和错题本中查看题目，确认能看到章节或试卷、知识点、原题号、文件名等定位信息。
5. 生成“自定义模拟组卷”，确认可按题型数量组卷，填空和简答只展示参考答案不计分。
6. 生成“单片机模拟考试（固定策略）”，确认原固定策略入口和 70 分计分逻辑仍可用。
