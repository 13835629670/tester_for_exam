# 刷题记忆工具

本地刷题工具，当前只支持：

- `.docx`
- `.doc`
- 单选题
- 判断题
- 顺序练习 / 乱序练习
- 错题本
- 全部预览
- 按科目导入和删除导入文件

## 推荐启动方式

```powershell
python server.py
```

然后访问：

```text
http://127.0.0.1:5175
```

用这种方式打开时，`.doc` 会调用本机 Word 转成 `.docx` 后再读取，通常比浏览器直接读取更稳。
老公式对象、MathType / Equation 公式和部分图片公式会尽量以图片形式保留，避免显示成 `EMBED Equation.3`。
Word 表格会保留为网页表格，适合真值表、卡诺图等内容。

也可以直接打开 `index.html` 使用；这种方式下只建议导入 `.docx`。
`.doc` 必须使用 `python server.py` 启动后的网页导入，否则浏览器无法可靠读取 Word 二进制文件。

## 推荐题库格式

```text
一、单选题（12 道）
聚类分析是将数据集合划分为多个类别的过程，聚类后同一类别中数据样本具有（ ）A. 较高的相似度B. 较低的相似度C. 相同的特征D. 不同的特征答案：A

二、判断题（3 道）
聚类分析是一种有监督学习方法。（ ）答案：错误
```

题型标题不是必须的，但每道题需要包含 `答案：`，单选答案使用 `A/B/C/D`，判断答案使用 `正确/错误` 或 `对/错`。

## 说明

本工具不保存数据，刷新页面后本次导入的科目、题目、错题都会清空。
导入错误的文件可以在左侧“已导入文件”中删除。
