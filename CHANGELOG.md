# 更新日志

## v0.1.2

- 新增 IndexedDB 本地缓存：题库、图片、错题、已答记录和练习进度会自动保存。
- 新增缓存导出 / 导入：可在电脑端导入含图片题库后导出 JSON，再导入到手机网页继续刷题。
- 新增跳题和清除进度功能，方便按题号继续练习。
- 保持当前版本仅支持 `.docx` / `.doc`，未重新加入 PDF 导入。

## v0.1.1

- 修复 `选项A）`、`选项B）` 这类 Word 题库格式无法识别为单选题的问题。
- 修复图片公式 token 被选项正则误改成 `[[IMG. ...]]`，导致图片无法显示并出现异常选项的问题。
- 修复包含大图片的题目因 base64 内容过长被误判为异常题块的问题。
- 为页面脚本增加版本号并禁用本地服务缓存，减少浏览器继续加载旧解析逻辑的情况。

## v0.1.0

- 初始版本：支持按科目导入 `.docx` / `.doc` 题库。
- 支持单选题、判断题、顺序练习、乱序练习、错题本和全部预览。
